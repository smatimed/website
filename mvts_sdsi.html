MONITORING OF STAFF MOVEMENTS

  • Allows the monitoring of staff movements: Annual leave, special leave, exit vouchers, sick leave, recoveries.

Technologies used:

  • Python, Django
  • Django's framework: rest_framework to create RESTful APIs (Web APIs)
  • Package used (not part of Django): widget_tweaks
  • Modules used (not part of Python): xhtml2pdf, openpyxl
  • HTML, CSS, Bootstrap, JavaScript, XHR / Fetch
  • MariaDB Database
  • NGINX (for deploiement)
